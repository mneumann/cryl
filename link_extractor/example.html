Projects

Here are some of my own projects listed (many people have contributed to them).

Ruby/DBI

A database independent interface to relational databases, similar to Perl's DBI but for Ruby. Supported databases include: DB2, Oracle 7-9i, PostgreSQL, MySQL, FrontBase, ODBC, ADO, Interbase, mSQL, SQLite, SQLRelay, Remote/Proxy (TCP/IP).

XML-RPC for Ruby

XML-RPC is a specification for Remote Procedure Calls (RPC) over HTTP, using XML as transmission format. This is an implementation for Ruby.

XSLT for Ruby

XSLT4R is an EXtensible Stylesheet Language for Transformations (or XSLT in short) processor written in pure Ruby. It's extensible through Ruby methods, but has rest in a prototype state (despite, it's quite usable).

ruby-ggi

GGI binding for Ruby.

Guugelhupf (de)

Project G.H. or Guugelhupf is my implementation of a simple search engine (index generation + query) in MLton (Standard ML). It is a result from a study task in computer science at the 4th semester. Have a look at some source codes. It was my first big program written in ML.
